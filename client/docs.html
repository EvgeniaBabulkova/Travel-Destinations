API Endpoints

Here are the main API endpoints used in our project:

TravelRoutes

Method Path Description
GET /api/travel Get all travel destinations.
GET /api/travel/:id Get a travel destination by ID.
POST /api/travel Create a new travel destination.
PUT /api/travel/:id Update a travel destination by ID.
DELETE /api/travel/:id Delete a travel destination by ID.

UserRoutes

Method Path Description
GET /api/users Get all users.
GET /api/users/:id Get a user by ID.
POST /api/users Create a new user.
PUT /api/users/:id Update a user by ID.
DELETE /api/users/:id Delete a user by ID.
